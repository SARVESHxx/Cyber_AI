ABOUT THIS PROJECT

        

          CyberMind is an open-source AI-powered threat intelligence tool.
          It helps security analysts, students, and researchers quickly triage and understand security data.
        

        
WHAT IT ANALYZES

        
    
          
  • ⟩ Server & access logs (Apache, Nginx, AWS, etc.)
    • 
          
  • ⟩ IP reputation & geolocation context
    • 
          
  • ⟩ Source code — OWASP Top 10, SQL injection, XSS, RCE
    • 
          
  • ⟩ File hashes — context and known malware patterns
    • 
          
  • ⟩ Email headers — phishing and spoofing detection
    • 
          
  • ⟩ Network traffic patterns
    • 
          
  • ⟩ CVE descriptions and severity assessment
    • 
        

        
TECH STACK

        
    
          
  • ⟩ Frontend: Vanilla HTML/CSS/JS (no frameworks needed)
    • 
          
  • ⟩ AI: Anthropic Claude API (claude-sonnet-4-20250514)
    • 
          
  • ⟩ Storage: localStorage for history
    • 
          
  • ⟩ Fonts: Google Fonts (Orbitron, Rajdhani, Share Tech Mono)
    • 
        

        
DISCLAIMER

        

          This tool is for educational and defensive security purposes only.
          AI analysis should be used as a supplement, not a replacement, for proper security tools and expert review.
          Never paste sensitive production data or credentials.
